# Global Superstore Sales Dashboard

## Overview
This Power BI dashboard analyzes sales, profit, orders, market performance, and category trends using the Global Superstore dataset.

# Key Metrics
- Total Sales: ₹47,44,691
- Total Profit: ₹6,63,779
- Total Orders: 178,312

## Key Insights
- Technology generated the highest sales.
- APAC was the top-performing market.
- Sales showed consistent growth from 2011 to 2014.
- Office Supplies and Furniture contributed significantly to revenue.
